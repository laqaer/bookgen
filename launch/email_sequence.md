# Email Sequence

## Sequence Structure

1. Problem framing: why this book exists.
2. Method framing: how the argument avoids overreach.
3. Excerpt mail: one strong passage + takeaway.
4. Case-study mail: applied institutional analysis.
5. Launch announcement.
6. FAQ mail: objections and boundaries.
7. Social proof mail: selected reader responses.
8. Audiobook + extended format mail.
9. Offer ladder mail: newsletter/course/consulting pathways.

## Voice Rule

Each email should sound like a briefing memo: clear stakes, bounded claims, practical next step.

## Core CTA

Read, share, and apply one framework from Shadows and Structures in a live public debate context.
