# Interior Layout Specs

## Trim and Format

- Primary trim candidate: 6 x 9 trade paperback.
- Alternate hardback trim if required by distributor constraints.

## Typography

- Body font: readable serif at 10.5-11 pt.
- Heading font: clean sans-serif with clear hierarchy.
- Line spacing tuned for dense nonfiction readability.

## Layout Rules

- Keep chapter openers uncluttered.
- Use pull quotes sparingly and only with verified references.
- Maintain consistent heading levels and citation placeholder style.

## Production QA

- Widow/orphan checks.
- Running headers aligned by chapter.
- Grayscale contrast pass for print reliability.
